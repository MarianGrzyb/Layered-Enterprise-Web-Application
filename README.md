__spring__: Spring Framework, Spring Boot, Spring Data

__spring-MVC__: Spring MVC, RESTful web services

__microservices__: decomposing a monolithic application into stand-alone modules using a microservices architecture

__angular__: Angular Framework, consuming microservices
